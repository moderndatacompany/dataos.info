# Databases supported by Superset

In DataOS, Superset supports the following databases:

- Apache Druid
- Apache Hive
- Apache spark SQL
- Aurora MySQL (Data API)
- Aurora PostgreSQL (Data API)
- ElasticSearch (OpenDistro SQL)
- ElasticSearch (SQL API)
- Google BigQuery
- MySQL
- PostgreSQL
- Presto
- SQLite
- Trino
- Other
