Welcome to dataos-pyflare's documentation!
==========================================
**Dataos-Pyflare** Dataos-PyFlare is a powerful Python library designed to simplify data operations
and interactions with the DataOS platform and Apache Spark. It provides a convenient and efficient
way to load, transform, and save data.
For more information and releases refer `Dataos-Pyflare <https://pypi.org/project/dataos-pyflare/>`_

.. note::

   This project is under active development.
