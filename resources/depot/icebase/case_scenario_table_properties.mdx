## List Properties

To obtain the list of properties, execute the following command within the DataOS.

**Command**

```bash
dataos-ctl dataset properties -a ${{udl}}
```

[](https://dataos.info/resources/depot/icebase/case_scenario_table_properties/#__codelineno-0-1)**Example**

```bash
dataos-ctl dataset properties -a dataos://icebase:retail/city
```

## Add Properties

To add a single property the below code can be used.

**Command**

```bash
dataos-ctl dataset add-properties -a ${{udl}} \
-p "${{property-name}}:${{property-value}}"
```

[](https://dataos.info/resources/depot/icebase/case_scenario_table_properties/#__codelineno-3-1)To add multiple properties at the same time, use:[](https://dataos.info/resources/depot/icebase/case_scenario_table_properties/#__codelineno-4-1)

```bash
dataos-ctl dataset add-properties -a dataos://icebase:retail/city \
-p "${{property-name}}:${{property-value}}" \
-p "${{property-name}}:${{property-value}}"
```

## Remove Properties

To remove a property, the following command can be used.

**Command**

```bash

-p "${{property-name}}" \
-p "${{property-name}}"
```
